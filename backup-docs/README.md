## 📋 目录说明

本目录存放历史文档的备份快照，主要用于：

1. **历史参考**：保留文档的历史版本，便于追溯变更
2. **恢复参考**：如需恢复某个历史版本的文档，可在此查找
3. **对比分析**：对比当前文档与历史版本的差异

## 📁 目录结构

- `eeecfe7back/` - 基于commit hash `eeecfe7` 的文档快照
  - 包含完整的文档结构备份
  - 包括：specifications、knowledge-base、work-plans等目录
- `docs/` - 部分文档备份

## ⚠️ 重要提示

1. **不要直接使用**：这些是历史备份，当前文档以 `docs/` 目录为准
2. **仅供参考**：如需参考历史版本，请先确认当前文档状态
3. **避免混淆**：不要将备份文档误认为是当前文档
