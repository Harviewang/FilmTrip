## 🚀 快速使用指南

### 1. 会话开始时
```markdown
复制"需求分析模板A" → 填写具体信息 → 提交给用户确认
```

### 2. 方案设计时
```markdown
复制"技术方案设计模板B" → 详细设计 → 使用"用户确认模板C"获得确认
```

### 3. 任务执行时
```markdown
使用"任务分配模板D"分配工作 → 各Agent使用"自查报告模板E"提交成果
```

### 4. 质量审查时
```markdown
@reviewer使用"质量审查模板F" → 用户使用"用户验收模板G"验收
```

### 5. 项目总结时
```markdown
@documenter使用"项目总结模板H" → 归档到知识库
```

---

**⚠️ 重要提醒**: 
- 所有模板都是强制使用的标准格式
- 每个阶段必须完整填写对应模板
- 模板内容要求真实、准确、详细
- 所有模板都要保存到项目文档库中
