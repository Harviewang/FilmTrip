## 🔧 使用方法

### 1. 选择模板
根据当前阶段和工作类型选择对应的模板

### 2. 复制模板
```bash
# 创建新的记录文件
cp template-a-demand-analysis.md logs/demand-analysis-[YYYY-MM-DD].md

# 或者直接复制内容使用
```

### 3. 填写信息
按照模板中的提示填写具体信息

### 4. 提交确认
完成后提交给相关人员确认或审核

### 5. 归档保存
完成后保存到对应的logs目录中
